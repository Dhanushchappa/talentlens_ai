# TalentLens AI

TalentLens AI is a machine learning-powered resume screening platform built with Streamlit.

## Features

* Resume parsing (PDF/TXT)
* ATS score calculation using TF-IDF + cosine similarity
* Candidate role prediction
* Skill gap analysis
* Resume ranking dashboard
* CSV export for HR analytics

## Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* Plotly
